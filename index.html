Syntactically Awesome Style Sheets (SASS)

What is SASS?

SASS (Syntactically Awesome Stylesheets) is a CSS pre-processor that lets you use variables, mathematical operations, mixins, loops, functions, imports, and other functionalities that super powers your CSS.

You can use variables.

A variable allows you to store a values and to reuse the variables whereever you want in you SCSS

$blue: #255271;
$orange: #ffa500;

$primary: $blue;
$secondary: $orange;

$content-heading-color: $secondary;
$link: $secondary;

Now you can use the color variable to style elements.

h1, h2, h3, h4, h5, h6 {
    color: $content-heading-color;
}            

.button {
    background-color: $primary;
}

a {
    color: $link-color;
}

Let's reverse the colors.

$blue: #255271;
$orange: #ffa500;

$primary: $orange;
$secondary: $blue;

$content-heading-color: $secondary;
$link: $secondary;

You can nest style rules.

.content {
  display: block;

  p {
    margin-top: 1rem;
  }

  ul {
    list-style: none;
  }
}

This will compile into:

.content {
  display: block;
}

.content p {
  margin-top: 1rem;
}

.content ul {
  list-style: none;
}

You can use mixins.

@mixin button($background: $primary, $color: $light) {
  background-color: $background;
  color: $color;
};

Include the mixin wherever you want.

.content {
  .button {
    @include button($purple, $yellow);
  }
}

This will compile into:

.content .button {
  background-color: #800080;
  color: #ff0000;
}

You can use built-in functions.

.content {
  .button {
    @include button(darken($purple, 10%), $yellow);
  }
}

This will compile into:

.content .button {
  background-color: #4d004d;
  color: #ffff00;
}

You can use partials.

@import "partials/functions";
@import "partials/variables";
@import "partials/mixins";
@import "partials/base/all";
@import "partials/components/all";
@import "partials/controls";
@import "partials/button";
@import "partials/container";
@import "partials/columns";
@import "partials/section";
@import "partials/content";
@import "partials/title";
@import "partials/form/all";
@import "partials/footer";
